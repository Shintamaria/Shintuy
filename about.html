About Me

Saya pernah memiliki cita-cita menjadi seorang front end developer dan saya juga pernah rajin belajar bahasa pemrograman dasar secara otodidak mulai dari HTML dan CSS kemudian sempat lanjut ke dasarnya javascript dan disitu saya mulai menemukan kesulitan dan akhirnya stuck dan memilih untuk fokus ke hobi bermain ML (mobile legend). ML merupakan game online yang dimainkan secara tim. Dalam satu tim terdiri atas 5 orang dengan durasi main 15-20 menit untuk 1 ronde. saya bermain ML sejak kelas 8 SMP tepatnya pada tahun 2018 sampai saat ini. saya memiliki hero favorit yaitu dyrroth, dyrroth saya sudah menenpati peringkat 5 besar di kecamatan kesamben dan masuk peringkat 20 besar di kabupaten blitar. Awal nya saya main ML hanya untuk pelampiasan emosi tapi semakin lama malah jadi emosi kalau tidak main ML. sudah 4 tahun saya memainkannya, itu membuat saya candu. Pada satu waktu saya pernah kepikiran dan ingin berhenti memainkan ML karena semakin kesini saya merasa sulit untuk mengontrol emosi apalagi saat pertandingannya tidak memuaskan, tapi setelah saya coba ternyata itu sangat sulit, semakin saya melupakannya semakin saya ingin memainkannya dan jika satu hari tanpa main ML saya menjadi tidak fokus untuk melakukan suatu hal atau kegiatan lain.
